Java

Java or Java is an object-oriented programming language developed by Sun Microsystems and launched in 1995 as part of the Java platform. Subsequently, many other implementations appeared, including GNU, Microsoft, IBM, Oracle, and other technology vendors. The source code, written in Java, is not compiled to machine code for a specific microprocessor, but is compiled to an intermediate language - the so-called bytecode. The byte code is not run for direct execution by the microprocessor, but is executed by its analogue - a virtual processor called Java Virtual Machine. Such a realization has its advantages and disadvantages.
